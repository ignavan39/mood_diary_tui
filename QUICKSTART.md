# 🚀 Quick Start Guide

## За 5 минут до первой записи

### 1. Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/ignavan39/mood-diary.git
cd mood-diary

# Установите зависимости
go mod download

# Соберите приложение
make build
# или
go build -o mood-diary ./cmd/mood-diary

# Запустите
./mood-diary
```

### 3. Создайте первую запись

**Шаг 1: Выберите "Записать настроение"**
```
Нажмите Enter на первом пункте меню
```

**Шаг 2: Выберите уровень настроения**
```
Как вы себя чувствуете сегодня?

😢 😞 😔 😕 😐 😶 🙂 😊 😄 😁 🤩
                    ^

😊 Хорошо (7/10)

←/→: Изменить • Enter: Далее • Esc: Назад
```

Используйте стрелки `←` и `→` для выбора уровня от 0 до 10.

**Шаг 3: Добавьте заметку (необязательно)**
```
Добавьте заметку (необязательно):

╭─────────────────────────────────────────────────────╮
│ Отличный день! Закончил проект_                     │
╰─────────────────────────────────────────────────────╯

Enter: Подтвердить • Esc: Назад
```

**Шаг 4: Подтвердите запись**
```
Подтвердите запись:

╭──────────────────────────────────────╮
│ Настроение: 😊 Хорошо (7/10)        │
│ Заметка: Отличный день! Закончил...  │
│ Дата: 01.04.2026                     │
╰──────────────────────────────────────╯

y/Enter: Сохранить • n/Esc: Отмена
```

Нажмите `y` или `Enter` для сохранения.

## Горячие клавиши

### Глобальные
- `Ctrl+C` - Выход из приложения (в любом месте)
- `q` - Выход / Возврат в главное меню
- `Esc` - Отмена / Шаг назад

### Главное меню
- `↑` / `k` - Вверх
- `↓` / `j` - Вниз
- `Enter` - Выбрать

### Запись настроения
- `←` / `h` - Уменьшить уровень
- `→` / `l` - Увеличить уровень
- `Enter` - Подтвердить / Далее
- `Esc` - Отмена / Назад
- `y` / `n` - Да / Нет (при подтверждении)

### Статистика
- `←` / `h` - Предыдущий период
- `→` / `l` - Следующий период
- `r` - Обновить данные
- `Esc` - Вернуться в меню

### История
- `↑` / `k` - Вверх
- `↓` / `j` - Вниз
- `Enter` - Редактировать запись
- `r` - Обновить список
- `Esc` - Вернуться в меню

## Типичные сценарии использования

### Ежедневная запись

**Утренний ритуал:**
```bash
# Запустите приложение
./mood-diary

# Нажмите Enter (Записать настроение)
# Выберите уровень настроения
# Добавьте заметку о планах на день
# Подтвердите запись
```

**Вечерняя рефлексия:**
```bash
# Откройте историю (пункт 3)
# Посмотрите на вашу запись за сегодня
# Вернитесь в статистику (пункт 2)
# Проанализируйте тренд за неделю
```

**Linux (systemd user service):**
```bash
# Создайте ~/.config/systemd/user/mood-diary-reminder.service
[Unit]
Description=Mood Diary Reminder
After=graphical-session.target

[Service]
Type=oneshot
ExecStart=/usr/bin/notify-send "Mood Diary" "Не забудьте записать настроение!"

[Install]
WantedBy=default.target

# Создайте таймер для запуска каждый вечер в 21:00
# ~/.config/systemd/user/mood-diary-reminder.timer
[Unit]
Description=Mood Diary Reminder Timer

[Timer]
OnCalendar=daily
OnCalendar=21:00
Persistent=true

[Install]
WantedBy=timers.target

# Включите таймер
systemctl --user enable mood-diary-reminder.timer
systemctl --user start mood-diary-reminder.timer
```

### Bash alias

```bash
# Добавьте в ~/.bashrc или ~/.zshrc
alias mood='cd ~/mood-diary && ./mood-diary'
alias mood-stats='cd ~/mood-diary && ./mood-diary -screen=stats'
alias mood-history='cd ~/mood-diary && ./mood-diary -screen=history'

# Теперь можно просто набрать:
mood
```

### Backup данных

```bash
# Создайте скрипт ~/bin/backup-mood.sh
#!/bin/bash
DATE=$(date +%Y%m%d)
cp ~/.mood-diary/mood_diary.db ~/backups/mood_diary_$DATE.db
echo "Backup created: ~/backups/mood_diary_$DATE.db"

# Сделайте исполняемым
chmod +x ~/bin/backup-mood.sh

# Добавьте в crontab для автоматического backup
crontab -e
# Каждый день в 23:00
0 23 * * * ~/bin/backup-mood.sh
```
